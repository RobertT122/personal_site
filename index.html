BoardGameNerd

 In BoardGameNerd you can explore a ranked list of board games sorted by user reviews, filtered by category, and thoughtfully displayed with game art and average score. A user can visit the site and see relevant information on the games they are interested in at a glance. Need to know the number of players? We got you covered. You are also able to leave reviews of the games you have played and view other users comments and ratings. If your favorite game is not featured on the site, fear not! You are able to upload your own game to BoardGameNerd with our intuitive submission form.
 BoardGameNerd is built using Ruby on Rails as the MVC framework with PostgreSQL as the general purpose database, while images are stored separately using AWS S3 buckets. The frontend is designed with React using hooks and functional components.

groupThink

Group think is a crowd sourced question and answering application that focuses on getting users gut reactions. The app allows you to ask a "Yes" or "No" questions to other users and get information on how people respond. Users will be given time limits to respond to questions, and receive rewards or penalties based on whether they answered as part of the majority. The function of this app is to apply social pressure and time pressure to users answering questions and provide feedback on their decisions.
groupThink was built on the MERN stack using Node.js, Express.js, and MongoDB for the backend, and React and Redux for the frontend.

Slime Slasher

  Slasher is a fast paced reaction game with a “rock-paper-scissors” style mechanism. In Slime Slasher you race against the clock to select the right attack, if you hit, you get points! If you miss, you lose a life (be careful, you only have three!) Beat your high score, challenge your friends, and enjoy the sometimes frustrating experience of battling slimes.
  Slasher is designed with a mobile first layout and input for maximum comfort and readability. It uses vanilla js and canvas elements to create clean animations and responsive controls.
  The game also contains accessibility features for those who have red green colorblindness, or those who just want a different look!
